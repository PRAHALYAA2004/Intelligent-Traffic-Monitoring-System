# SETUP

## Setup Guide
### Prerequisites
- Install **Node.js**, **MongoDB**, and **Docker**.

### Steps
1. Clone the repository:
   ```sh
   git clone https://github.com/PRAHALYAA2004/Intelligent-Traffic-Monitoring-System.git
   ```
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the application:
   ```sh
   npm start
   ```
